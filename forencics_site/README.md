# Forencics Challenge 2023 team6

Website for presentation. 

## Run

### Step 1. Install Hatch](https://hatch.pypa.io)

```bash
# preferred
pipx install hatch

```

### Step 2.  `flask_secret_key` in file `config.yaml`

### Step 3. Run server:

```bash
hatch run server -c config.yaml
```
